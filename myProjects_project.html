FRIDAY, 03 SEPT 2009
DOALOT Phase 1 Tech Spaces
1 Comment Last by Imran Rashid on Mon, 6, Jul at 12:50pm
In the meeting this morning both, Frank and I, agreed that we have covered everything for phase 1 of the project. The final version for phase 1 is attached. -Imran.

Posted by Imran Rashid | 1 comment

Meeting Minutes
Participants: Frank and Imran.
Following points were discussed in today’s meeting:
  • The design guide would be ready I one or two days.
  • Frank is meeting with Abel this week and will be sending the updates on design as they are make available to him.
  • Imran is to post he updated project plan.
  • The front page that is shown in the PowerPoint file is not be implemented in the solution. The homepage of the site would be like slide no 2 in the file.
  • There was some duplicated information shown in the PowerPoint slides in their user profile section such as My projects was shown both in the main content area and in the right side bar. The right side bar would only contain the “Most recently added projects” list in the first version. In later versions, it could have additional tabs such as highest rated projects and Most talked about projects.
  • The picture of the project will be optional. In case a user does not upload the project picture, a default picture will be presented to the user on the project homepage an also on the search results page.
  • The search result page will be text based in the first version while in later version more views for search results page will be added such as the one shown on Dundas website and CN page shown in the PowerPoint file.

I can’t see what all the fuss is about. The characterization is so weak that in some places I cringed with embarrassment for Dan Brown, though I suspect he’s laughing all the way to the bank. The plot is OK, and the subject is interesting, but I recommend you try some of the 1980’s books he actually mentions during the story, if you’re looking for a better read. In my opinion, Brown’s writing is crude and without any finesse so I found it very hard to stick with it to the end. I did finish it though – it would have been an anti-climax if there’d been any previous climax worth mentioning.
